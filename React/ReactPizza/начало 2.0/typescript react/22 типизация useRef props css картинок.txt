типизация картинок который импортируются как модули
чтобы наш ts понимал какой тип у картинок мы создаем где нибудь папку @types
и внутри пишем luboeNazvanie.d.ts
и туда пихаем
declare module '*.svg/png.jpeg'{
	const content: any;
	export default content;
}

так же в нашем tsconfig.json мы дописываем
"include": ["src","src/@types"] - чтобы везде работали наши типы

если у нас есть переменная и это множество чего то то мы можем сделать свой тип
мы пишем type SortItem = {
	name:string;
	sortProperty:string;
};

далее мы пишем что наша переменная возвращает этот тип 
const sortList: SortItem[](квадратные скобочки нужны чтобы показать ts что тут много объектов  в массиве) = [
{name:'kok',sortProperty: 'kokprop'} - тут обязательно name и sortProperty, на другое ts будет ругаться
]

чтобы типизоровать useRef мы либо пишем в круглых скобочках null но только его
другие типы ts не принимает
либо пишем треугольные 
useRef<HTMLDivElement(то что мы пишем тут зависит от того к каком тэгу мы биндим ref в данном примере к div)>()


чтобы типизировать пропсы мы создаем тип в котором будет объект
type CategoriesProps = {
	value: number;


	onChangeCategory: any;
}
далее чтобы типизировать пропсы компоненты мы пишем после React.FC<CategoriesProps>

так же чтобы работили стили добавляем там где у нас svg png дописываем еще scss css и тд

если мы испольщуем какую то библеотеку и в ней есть типы то мы ищем в инете 
и скачиваем npm lodash types например
