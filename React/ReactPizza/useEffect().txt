useEffect() - это хук который позволяет заменить классовые компоненты 
в него входят методы жизненного цикла из классовых компонент
синтаксис

	React.useEffect(()=>{},[]) - передаваемый массив это передаваемое значение, если передаем массив то это componentdidmount
когда компонента только появится если в квадратные скобки чтото записать то при изменении чего то в них будет перезапускаться
логика в useEffect()
если прописать там return () =>{} то это ззначит выполни это если компонента умирает
