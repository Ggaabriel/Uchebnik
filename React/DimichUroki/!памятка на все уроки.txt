a. Чистая функция - детерминирована и не обладает side эффектом
b. Детерминированность - для одних и тех же входных данных один и тот же результат
c. Идемпотентность - при многократном вызове функции состояние будет изменено единожды
d. Имьютабельность - неизменяемость. То есть имьютабельный объект - объект,
 состояние которого не может быть изменено после создания
e. Side effect - возможность в процессе выполнения вычислений: читать, модифицировать значения глобальных переменных,
 осуществлять операции ввода-вывода... Например, взаимодействие с БД, с глобальными переменными, HTTP-запросы, случайные числа.
