props - нужен например чтобы сделать 1 тэг и у него менять например текст
это параметр в функции компоненте, внутри которого хранится наш атрибут который мы вписываем в jsx разметке
пример
делаем тэг <Footer message='i am footer' />
мы сделал атрибут message теперь в нашем prods хранится ключ message со значением i am footer 
то есть prods = {
	message:`i am footer`
}
теперь чтобы его использовать в компоненте мы пишем
const bebra = (prods) => {
 	return(
		{props.message}
	)
}
теперь у нас в index.html будет сообщение в футере
