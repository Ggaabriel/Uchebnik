задаем ссылкам в href = /(название компоненты)
Для перехода с одной разметки на другую нужен модуль route
чтобы его скачать нужно прописать npm install react-router-dom -save(это,
	чтобы сохранить изменения в package.json для клонирования в в гит)

чтобы переходить по разметкам есть особый тег <Route element={тэг компоненты} />

но это все не работает без обрамления, чтобы обрамленить пишем <BrowserRouter> в ретурн

сейчас у нас роуты не видят адресную строку
нам нужно прописать им path='/название компоненты'
например:
<Route path="/profile" element={<Profile />}/>

с новым синтаксисом теперь нужно засовывать все роуты в тэг Routes
например:
<Routes>
            <Route path="/profile" element={<Profile />}/>
            <Route path="/dialogs" element={<Dialogs />}/>
        </Routes>
