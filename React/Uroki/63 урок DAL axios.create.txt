DAL - это отдельный от ui и bll раздел связанный с запросами на сервер, он общается с bll а bll с ui но в нашем примере
ui будет общатся с dal 

делаем отдельную папку и файл с js форматом пихаем туда наши get запросы но не then ответы.

для удобства создаем объект instance который хранит в себе наши урлы авторизацию и апи ключи
создаем через axios.create({})
и пускай дальше мы будем обращаться в запросах не к axios а к instance
для удобства делаем один объект и пихаем все наши запросы туда и делаем их методами
