для flux архитектуры мы очень нагло поступаем создавая в UI объект поэтому мы будем пользоваться action creator, action type
-action creator это функция в state которая создает в нужном нам месте объект для dispatch 
она может принимать или не принимать параметр 
	она возвращает объект с обязательным type 
	например: let addPostActionCreator = () => ({type: 'ADD-POST'})
мы их создаем в state и экспортируем в нужную компоненту

функция должна использовать только то что ей приходит в параметрах но только когда речь о данных от которых что-то зависит

-action type это константа которая хранит в себе значения будущего type
	например: const UPDATE_NEW_POST_TEXT = 'UPDATE_NEW_POST_TEXT';
