Контекст - это такая система которая позволяет нам не прокидывать данные через пропсы а просто доставать их в любом месте
дочерней компоненты.
что бы сделать контекст надо сделать отдельный файл 
	импортим react from react 
	делаем переменную const store = React.createContex(null);
	экспортируем ее
дальше экспортируем наш контекст в нужную нам компоненту а именно в App
обхватываем ее 
<store.Provider value={/* некоторое значение */}>
	<app>
</store.Provider>
далее мы идем к нашему контейнеру и там в return пишем
<store.Consumer>
  {store =>() /* возвращает так же jsx разметку внутри другой компоненты */}
</store.Consumer>
